# 更新日志

## 1.1.1

- 修复任务到期提醒过度依赖 Windows 系统通知的问题。
- 增加应用内 Samguo 提醒条、任务栏闪烁和托盘气泡兜底。
- 新建、编辑任务后会立即检查提醒，不再只等待 30 秒轮询。

## 1.1.0

- 将界面重设计为“胶囊小屋”卡通治愈风格。
- 增加胶囊小助手、贴纸式任务卡片和更柔和的奶油色主题。
- 优化空状态、右侧看板、任务卡片和按钮的视觉层级。
- 增加今日便签台、任务卡胶带细节、彩色统计贴纸、操作成功提示和可折叠侧栏区块。
- 将星期、趋势提示和页面语言标记统一为中文体验。
- 增加 Samguo 专属卡通形象和右侧陪伴角色卡，让界面更有个人辨识度。
- Samguo 角色卡会根据空列表、逾期、今日任务、完成和未来计划自动切换文案、徽章和色调。
- 完成任务时增加“已收纳”贴纸印章和 Samguo 庆祝动效，批量完成也会触发盖章反馈。
- 默认列表改为今日优先排序，逾期、今天、置顶和高优先级任务会自然靠前显示。
- 优化空状态文案，并为新增任务加入轻量便签投递动效；智能输入支持“今天、明天、周末、高、中、低”。
- 删除任务支持 toast 撤销恢复，降低误操作成本。
- 重写 README，补充运行、打包、核心体验和数据能力说明。
- 将应用图标和托盘图标切换为 Samguo 卡通形象，增强产品记忆点。
- 增加 Samguo 开机小动画，启动时以轻量欢迎卡片进入主界面。
- 保留 v1.0.0 的功能结构，重点提升界面辨识度和使用亲和力。

## 1.0.0

- 将原项目重构为胶囊待办。
- 增加本地任务存储、提醒、托盘常驻和 JSON 导入/导出。
- 增加项目、标签、项目颜色、项目工具、智能输入和命令面板。
- 增加专注模式、项目分组、多选任务、批量操作和时间洞察面板。
- 增加自动备份、恢复备份、设置面板、关于面板和自定义应用图标。
